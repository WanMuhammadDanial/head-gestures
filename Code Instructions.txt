First time setup, run:
pip install -r requirements.txt

if there is 'SDK seems invalid' error, delete:
.idea file, which is generated by pycharm for each project


TO TRAIN DATA
1. Use main.py and modify line 39 to train said angle of gesture (train 1 at a time for breaks)
2. Once all angles are recorded, run the TrainStuff.py

List of angles:
	'neutral', 'up', 'down', 'left', 'right', 'tilt left', 'tilt right', 'shake head'

TO Test
1. Make sure to have a .h5 file in folder
2. Run the TestGesture.py
